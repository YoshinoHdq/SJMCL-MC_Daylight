## 简介

MC Daylight 是一款 SJMCL 扩展，在启动器首页实时显示你的 Minecraft 服务器时间与昼夜状态，卡片背景随游戏内太阳位置平滑过渡。

## 功能特色

- 🕐 实时时钟 — 现实时间与游戏时间并排显示
- 🌍 独立服务器时间 — 每台服务器独立追踪游戏 tick
- 🌙 月相显示 — 8 相月循环
- 📊 进度条 — 直观显示太阳在天空中的位置
- 🎨 色彩变换卡片 — 背景在 8 个昼夜色板间平滑插值
- 🖼️ 服务器图标 — 自动获取
- ➕ 多服务器支持 — 无限添加，各自独立设置

## 使用方法

1. 从 Releases 下载 .sjmclx 文件
2. 打开 SJMCL → 设置 → 扩展管理 → 导入
3. 在扩展设置页添加服务器并设置当前游戏时间

### 预设时间

| 按钮 | Tick | 说明 |
|------|------|------|
| 破晓 | 0 | 06:00 – 日出 |
| 清晨 | 2000 | 08:00 – 黄金时段 |
| 正午 | 6000 | 12:00 – 太阳当空 |
| 黄昏 | 12000 | 18:00 – 日落 |
| 夜晚 | 18000 | 00:00 – 深夜 |

### 技术说明

- Minecraft 以 20 ticks/秒 运行，一个游戏日 = 24000 ticks = 20 分钟现实时间
- 月相周期为 8 个游戏日
- 颜色在 8 个昼夜关键帧之间进行 RGB 插值

## 兼容性

- 最低启动器版本: 1.1.3
